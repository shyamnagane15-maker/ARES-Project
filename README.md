ARES is a high-performance mobile safety application designed to provide rapid emergency response and real-time security features. Built with a focus on reliability and speed, ARES serves as a digital guardian for users in critical situations.

#**_Key Features_**
Instant Emergency Alerts: One-tap triggers to notify emergency contacts and local authorities.

Real-time Location Tracking: Precise GPS integration to share live movements with trusted circles.

Secure Backend Architecture: Robust data handling to ensure user privacy and system uptime during crises.

User-Centric Interface: A clean, intuitive UI designed for high-stress environments where every second counts.

#**_Tech Stack_**
Frontend: React Native (Cross-platform iOS & Android)

Language: JavaScript / TypeScript

Backend: Firebase., Node.js

Design: Custom-designed logo and assets focused on a professional, tactical aesthetic.

#**_Installation and Setup_**

1. Clone the repository
2. Install the dependancies: cd ARES
                             npm install
3. Running the Application:  npx react-native run-android          =>For Android
                             npx react-native run-ios              =>For IOS
